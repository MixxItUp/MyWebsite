# 个人网站使用手册

## 项目结构

```
src/
├── assets/ // 存储项目所需的静态资源
│   ├── images/ // 图片资源
│   ├── videos/ // 视频资源
├── components/ // 项目的UI组件
├── views/ // 项目的视图组件
├── router/ // 路由配置
├── App.vue // 项目的根组件
├── main.js // 项目的入口文件
├── package.json // 项目的依赖配置
├── README.md // 项目的说明文档
```
